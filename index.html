FastLap AI

Formula 1 Lap Time Prediction using Machine Learning

About

FastLap AI is a machine learning project that predicts Formula 1 lap times using historical race data. The project focuses on understanding lap-time behavior and evaluating regression performance on real-world sports data.

Dataset

The dataset is sourced from the Formula 1 World Championship dataset (Kaggle). It includes lap-level information such as lap number, race position, and lap time recorded in milliseconds.

Model

A Linear Regression model was used as a baseline to predict lap times based on:

  • Lap Number
  • Race Position

The model achieved an MAE of ~11.5 seconds, highlighting the complexity of racing dynamics and real-world noise.
